What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents that represents the structure of a webpage as a tree of objects. It allows developers to interact with and manipulate HTML and XML documents, enabling dynamic content updates, event handling, and modifications to elements, attributes, and styles using scripting languages like JavaScript. The DOM provides a way to access and change the content and structure of a webpage without reloading it.

How do you select an element from the DOM?

You can select an element from the DOM using methods like getElementById('id') for selecting by ID, querySelector('.class') for selecting by class or other CSS selectors, and getElementsByTagName('tag') for selecting by tag name.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events on its child elements. It's useful because it improves performance by reducing the number of event listeners and allows handling dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, use setAttribute to change attributes and getAttribute to retrieve them. Modify inline styles with the style property, like element.style.backgroundColor = 'red', and manage CSS classes using classList.add() and classList.remove(). These methods allow dynamic changes to elements on your webpage.
